# Wan Multi Ref Stack

Packs many images into one `IMAGE` batch.

This is now the simple manual stacker. If you want drag and drop upload, image preview, frame-by-frame pinning, and a ready-made positions output, use `Wan Positioned Ref Gallery` instead.

Use this before `Wan VACE Multi-Ref To Video` for:

- middle keyframes
- extra person / object refs
- style refs

If image sizes do not match, the node can either:

- match the first image exactly
- crop everything to the first image
